## 📋 О проекте

OnyxChips - это полнофункциональное веб-приложение онлайн-казино с профессиональным дизайном, системой прогрессии и звуковыми эффектами. Проект включает систему аутентификации, игровые механики, систему уровней и опыта, ежедневные бонусы и адаптивный дизайн для всех устройств.

### ✨ Основные возможности

- 🎰 **Игровые автоматы (Slots)** - 3x3 слоты с анимациями, звуковыми эффектами и подсветкой выигрышных линий
- 🎮 **Система прогрессии** - уровни, опыт, ранги и награды за повышение уровня
- 🎁 **Ежедневные бонусы** - система стриков с увеличивающимися наградами
- 👤 **Аутентификация** - регистрация, вход, профиль пользователя с полной статистикой
- 💰 **Управление балансом** - отслеживание транзакций, выигрышей и игровых сессий
- 🎨 **Темная/Светлая тема** - переключение между темами с сохранением настроек
- 🔊 **Звуковые эффекты** - процедурная генерация звуков через Web Audio API
- 📱 **Адаптивный дизайн** - полная поддержка мобильных устройств и планшетов
- 🌐 **Мультиязычность** - поддержка русского и английского языков
- ⌨️ **Горячие клавиши** - управление игрой с клавиатуры (Space, ↑↓)
- 📊 **Статистика** - детальная статистика игр, выигрышей и прогресса

## 🗄️ Модели данных

### User (Пользователь)
Расширенная модель пользователя Django с игровой прогрессией:
- `username` - имя пользователя
- `email` - электронная почта
- `balance` - текущий баланс фишек
- `level` - текущий уровень игрока
- `experience` - накопленный опыт
- `rank_title` - игровое звание
- `total_games` - всего сыграно игр
- `total_wins` - всего выигрышей
- `total_losses` - всего проигрышей
- `biggest_win` - самый большой выигрыш
- `last_daily_bonus` - дата последнего ежедневного бонуса
- `daily_bonus_streak` - текущая серия ежедневных бонусов
- `sound_enabled` - включены ли звуки
- `language` - выбранный язык (ru/en)

### Game (Игра)
- `game_name` - название игры
- `description` - описание
- `is_active` - активна ли игра

### GameSession (Игровая сессия)
- `user` - пользователь
- `game` - игра
- `bet_amount` - размер ставки
- `win_amount` - выигрыш
- `balance_before/after` - баланс до/после
- `game_result` - результат (JSON)
- `created_at` - время создания

### Transaction (Транзакция)
- `user` - пользователь
- `amount` - сумма
- `transaction_type` - тип транзакции
- `comment` - комментарий
- `created_at` - время создания

## 🚀 Установка и запуск

### Требования
- Python 3.10+
- Django 6.0+
- SQLite (по умолчанию)

### Установка

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/Th3K3rn3l/Onyx-Chips.git
cd Onyx-Chips
```

2. **Создайте виртуальное окружение:**
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
```

3. **Установите зависимости:**
```bash
pip install django
```

4. **Примените миграции:**
```bash
cd onyxChips
python manage.py migrate
```

5. **Создайте суперпользователя:**
```bash
python manage.py createsuperuser
```

6. **Запустите сервер разработки:**
```bash
python manage.py runserver
```

7. **Откройте в браузере:**
```
http://127.0.0.1:8000/
```

**Доступные страницы:**
- Главная: `http://127.0.0.1:8000/`
- Игра Слоты: `http://127.0.0.1:8000/slots`
- Админ-панель: `http://127.0.0.1:8000/admin/`

## 🧪 Тестирование

### RTP-симуляция слотов

Запустите симулятор для проверки Return To Player:

```bash
cd onyxChips/main/tests
python rtp_simulator.py
```

Симулятор проведет 10,000 спинов и покажет:
- Общий RTP (должен быть 90-95%)
- Hit Rate (частота выигрышей)
- Статистику выплат
